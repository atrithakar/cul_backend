Upload a module

Guidelines for uploading modules

  • Make sure the repository is public
  • Make sure the repository contains a README.md file
  • Make sure the repository contains a LICENSE file

Folder structure should look like this:

        module_name
        |__1.0.0
        |   |__README.md
        |   |__LICENSE (optional)
        |   |__module.cpp or module.c
        |   |__module_info.json
        |__1.0.1
        |   |__README.md
        |   |__LICENSE (optional)
        |   |__module.cpp or module.c
        |   |__module_info.json
        and so on...
